## BRAND IDENTITY: GEAN
Premium hand care / personal care product brand. Clean, minimal, natural, and sophisticated. The aesthetic is organic luxury — not tech, not modern-gradient, not flashy.

---

## DESIGN SYSTEM (copy exactly from reference images)

### Color Palette
- Page background: #FFFFFF (white)
- Hero section: Dark lifestyle photo overlay (keyboard/desk scene) — use a dark semi-transparent overlay (#1a1a1a at 50%) over a background-color placeholder `#3d3028`
- CTA / primary button: Gold/amber `#C8A870`, pill-shaped with white text, hover slightly darker
- "BUY NOW" nav button: Dark charcoal `#2d2d2d`, rounded, white text
- Feature/accent section background: Sage green `#b8c9b5`
- Article/warm section background: Warm tan/brown `#a89070`
- Card background: White `#FFFFFF` with subtle rounded corners (border-radius: 16px)
- Body text: `#1a1a1a`
- Subtext / secondary: `#5a5a5a`
- Section divider / light bg: `#f5f0eb`

### Typography
- Font: `'Montserrat', sans-serif` — import from Google Fonts
- Headings: font-weight 300 (light), uppercase, wide letter-spacing (letter-spacing: 0.1em)
- Hero headline: font-size ~48px, weight 300, white, uppercase
- Section titles: font-size ~28–32px, weight 300, uppercase, centered
- Body: font-size 14–16px, weight 400, line-height 1.7
- Thai body text: use same font (Montserrat supports Latin; Thai fallback: `'Sarabun', sans-serif`)
- DO NOT use bold heavy headings

### Navigation (sticky, white background)
- Left: GEAN logo (text logo, charcoal, uppercase, light weight)
- Center: nav links — SHOP | OUR STORY | ARTICLE | CONTACT US — uppercase, small font-size, letter-spacing, no underline
- Right: cart icon (🛒) + "BUY NOW" dark pill button
- Border-bottom: 1px solid #eee

## GENERAL REQUIREMENTS

- Create one standalone HTML file per screen in `/ui-prototypes/` directory
- All CSS inline or in `<style>` tag — no external CSS files
- Import fonts: `<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@300;400;500&family=Sarabun:wght@300;400&display=swap" rel="stylesheet">`
- NO gradient backgrounds on page sections — use flat solid colors from the palette above
- NO bright/neon colors — keep the palette earthy, warm, minimal
- Responsive design (mobile-friendly)
- Navigation links between pages should work (relative href links)
- Use placeholder divs with background colors for images (not broken img tags)
- Image placeholders: background-color `#d4ccc2`, border-radius matching context
- All prototype files: home.html, shop.html, our-story.html, article.html, contact.html
- Replicate the section order and layout structure visible in the reference images as closely as possible
