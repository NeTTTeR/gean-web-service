Please implement a complete web application based on the specifications and requirements provided in the following documents:

1. **Requirements Document**: `docs/requirement.md`
2. **Technical Specification**: `docs/technical-spec.md`

**Implementation Requirements:**
- Implement ALL features specified in the requirements and technical specification documents
- Use the UI/UX screen prototypes located in `ui-prototypes/` as the visual design reference for all pages and components
- Ensure the implementation matches the design prototypes in terms of layout, styling, and user interface elements
- Follow the app-wide design tokens for consistency across all pages
- Generate well-structured, maintainable code following best practices

**Before starting implementation:**
1. Review the requirements document to understand all features needed
2. Review the technical specification to understand the architecture and technical approach
3. Examine all UI prototypes to understand the complete user interface design
4. Create a comprehensive implementation plan breaking down all features into tasks

**During implementation:**
- Ensure all pages use consistent styling based on the design tokens
- Implement responsive design for all components
- Follow the structure and patterns established in the existing codebase
